Developed a machine-learning and statistical analysis pipeline to predict resilience among home care workers using multimodal data collected from wearable sensors and digital cognitive assessments. The project integrates Ōura Ring physiological/sleep data, Stop-Go/PVT cognitive performance data, and Brief Resilience Scale scores to identify physiological and cognitive predictors associated with resilience.

The dataset combines repeated daily observations from multiple participants. In the machine-learning notebook, eight participant CSV files are combined into one dataset with 115 rows and 29 columns. The dataset includes sleep, physiological, activity, and cognitive variables such as lowest heart rate, total sleep duration, time in bed, restless periods, REM sleep duration, light sleep duration, deep sleep duration, HRV, sleep efficiency, average heart rate, average breathing rate, reaction time, steps, activity calories, and total calories.

The project also includes longitudinal feature engineering using rolling averages. A 5-day rolling window is applied by participant to generate rolling-average features for sleep and physiological variables such as lowest heart rate, total sleep duration, restless periods, REM sleep duration, sleep efficiency, average heart rate, average breathing rate, and average HRV.
